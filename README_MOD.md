PNW Plants - Mod Notes
======================

Summary
-------
This Vintage Story content mod adds Pacific Northwest plants. Included here is a single flower: Digitalis (foxglove).

Quick usage
-----------
- Install the `pnwplants` folder into your Vintage Story `Mods` directory.
- The flower block is registered as `pnwplants:flower-digitalis-purpurea`.
- Worldgen spawn is configured in `worldgen/blockpatches/flower.json`.

Developer notes
---------------
- Block type: `assets/pnwplants/blocktypes/plant/digitalis-purpurea.json`
- Shape: `assets/pnwplants/shapes/block/plant/flower/digitalis-purpurea/one-stalk.json`
- Textures: `assets/pnwplants/textures/blocks/plant/flower/digitalis-purpurea/digitalis-purpurea.png`
- Ensure texture atlas layout matches the UVs in the shape file.
